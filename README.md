# Bash Scripting Learning Repository

This repository contains **Bash scripts** created during my **Bash learning phase**.  
The goal of this repo is to practice, understand, and document core Bash scripting concepts used in Linux environments.

All scripts are written and tested on **Ubuntu (Linux)**.

---

##  Purpose

- Learn Bash scripting fundamentals
- Practice automation using shell scripts
- Understand Linux command-line tools and scripting patterns
- Build reusable scripts for daily development and system tasks
